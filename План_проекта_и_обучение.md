# План проекта Bug Reports Platform + Обучение тестированию

## ЧТО МЫ СТРОИМ (обзор)

**Платформа для создания багрепортов** — инструмент, который помогает тестировщику быстро оформить найденную ошибку в стандартный отчёт и отправить её в трекер (Jira, YouTrack и т.д.).

---

## ТЕХНОЛОГИИ (что будем использовать)

| Часть | Технология | Зачем |
|-------|------------|-------|
| Веб-фронт | HTML, CSS, JavaScript | Простота, не нужна сборка |
| Стили | Можно Tailwind или чистый CSS | Быстрое оформление |
| PDF | jsPDF или html2pdf | Генерация PDF из формы |
| Скриншот | html2canvas + getUserMedia | Захват экрана/области |
| Интеграции | Fetch API | Запросы к Jira, YouTrack |
| Telegram | Bot API | Бот |
| Десктоп | Electron | Упаковка в EXE |
