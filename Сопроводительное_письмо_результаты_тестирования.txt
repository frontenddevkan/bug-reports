НАЙДЕННЫЕ ПРОБЛЕМЫ (7 дефектов):

Критические:
1. Отсутствуют заголовки безопасности (X-Frame-Options, X-Content-Type-Options, Strict-Transport-Security, Content-Security-Policy). Рекомендуется добавить в конфигурацию nginx для защиты от clickjacking, XSS и MIME-sniffing атак.

2. Нет контента для пользователей без JavaScript — в исходном HTML только пустой div#root. При отключенном JS страница пустая, возможны проблемы с индексацией. Рекомендация: SSR или базовый контент в HTML.

Высокий приоритет:
3. sitemap.xml возвращает HTML вместо XML — поисковые системы не могут корректно прочитать карту сайта. Нужно настроить сервер для отдачи XML.

4. Отсутствует meta description — снижается контроль над отображением в поисковой выдаче.

5. Некорректная кодировка в title и manifest.json — вместо «Лига Цифровой Экономики» отображаются кракозябры. Влияет на закладки и добавление на главный экран мобильных устройств.

Средний приоритет:
6. Отсутствует заголовок Cache-Control — может негативно влиять на производительность и кэширование.
