# Bug Reports Platform — Design System & Context

## Project
Обучающая платформа-симулятор для QA-тестировщиков. Одностраничное приложение (vanilla HTML/CSS/JS, без фреймворков).

## File Structure
- `bug-reports-platform/index.html` — единственная HTML-страница
- `bug-reports-platform/app.js` — вся логика: canvas-анимация гексов, SDLC, квизы, роадмап, чеклисты
- `bug-reports-platform/styles.css` — все стили

## Visual Design — ЗАФИКСИРОВАННЫЙ СТИЛЬ (commit bef71ee)

### Цветовая палитра
- **Фон страницы**: `#000208` — почти чёрный с лёгким синим оттенком
- **Фон панелей**: `rgba(6, 14, 30, 0.75)` — тёмно-синий полупрозрачный
- **Бордеры панелей**: `rgba(40, 150, 220, 0.25)` — тонкая линия 0.5px
- **Glow бордеров**: `box-shadow: 0 0 8px rgba(40, 150, 220, 0.06)` — мягкое свечение
- **Текст основной**: `#e5e7eb`
- **Текст приглушённый**: `#9ca3af`
- **Акцент**: голубой диапазон `rgba(40-80, 120-180, 200-255, ...)`

### Фон: гексагональная сетка (canvas)
- **Тип**: pointy-top гексагоны, `HEX_RADIUS = 15px`
- **Прозрачность сетки**: очень низкая — glass-заливка 0.012, рёбра 0.05, блики 0.018
- **Сетка едва видна** — это фон, не акцент. НЕ увеличивать прозрачность
- **Огоньки**: 30 штук, движутся по рёбрам, `coreR = 1.5`, приглушённые
- **Диагональный fade**: от левого верха (ярче) к правому низу (тусклее)

### Свечение гекс при проходе огонька
- **Плавное нарастание**: `GLOW_RISE = 0.03` за кадр (lerp current→target)
- **Плавное угасание**: `GLOW_FADE = 0.012` за кадр
- **Распространение**: на ~2 кольца соседних гекс (`SPREAD_RADIUS = HEX_RADIUS * 3.5`)
- **Затухание по расстоянию**: квадратичное (`distFactor^2`)
- **Свечение граней**: 3 слоя (широкий glow 3.5px → средний 1.8px → ядро 0.8px)
- Ближняя гекса загорается первой, соседние следом с меньшей интенсивностью
- Всё плавно гаснет когда кружок удаляется

### Бордеры панелей
- Стиль: **чистые тонкие линии** с мягким glow
- НЕ использовать: ломаные/зубчатые линии, SVG circuit-board, анимированные бордеры
- Многослойность: тонкий border + box-shadow glow
- border-radius: 0 (убраны скругления через `!important`)

### Роадмап
- Вместо кружков — **гексагоны** (`clip-path: polygon(...)`)
- Два слоя: `::before` (внешний бордер-гекс), `::after` (glass-блик)
- Состояния: done (синий, drop-shadow), current (пульсация), default (тёмный)

### Шапка
- Полупрозрачный тёмный фон: `rgba(8, 20, 50, 0.45)` → `rgba(4, 10, 30, 0.55)`
- Гексы фона просвечивают сквозь шапку
- Тонкий разделитель снизу: `border-bottom: 0.5px solid rgba(40, 140, 220, 0.18)`

### Общие принципы
1. **Минимализм**: всё приглушённо, фон не отвлекает от контента
2. **Глубокий тёмно-синий тон**: не фиолетовый, не бирюзовый — синий (#0a1e3a-диапазон)
3. **Свечение мягкое**: никаких резких ярких вспышек
4. **Анимации плавные**: lerp, не мгновенные переключения
5. **Бордеры тонкие**: 0.5px, не 1px+
6. **Фон панелей тёмный**: rgba(6,14,30,...), не rgba(15,23,42,...) — ещё темнее

## Tech Notes
- Canvas рендерится на 30 FPS (`FRAME_MS = 1000/30`)
- Два canvas-слоя: `gridLayer` (статика, перерисовка при resize) + основной (анимация)
- `globalCompositeOperation: 'screen'` для аддитивного наложения свечения
- Маршруты огоньков: 80 сегментов, граф смежности вершин гексов
